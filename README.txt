Essence by Siva - README

Files included:
- index.html    --> Main website file
- images/       --> Folder for your perfume images (empty)

How to preview locally:
1. Extract the ZIP file.
2. Open the folder 'Essence_by_Siva'.
3. Double-click 'index.html' to open in your browser (Chrome/Edge/Firefox).

How to add your images:
1. Place your realistic perfume images in the 'images' folder.
2. Name them exactly: perfume1.jpg, perfume2.jpg, perfume3.jpg, perfume4.jpg, perfume5.jpg, perfume6.jpg
3. Refresh the browser to see them appear on the page.

How to host on GitHub Pages:
1. Create a GitHub account (if you don't have one): https://github.com
2. Create a new repository named 'EssenceBySiva' (or any name you like).
3. Upload the contents of the 'Essence_by_Siva' folder (index.html and images/).
4. Go to Settings -> Pages -> Source -> select 'main' branch and folder '/' (root), then Save.
5. Wait 1-2 minutes. Your site will be available at: https://yourusername.github.io/EssenceBySiva/
